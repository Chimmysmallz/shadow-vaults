# Changelog

All notable changes to Shadow Vaults are documented here.

## [1.0.0]
First public release.

### Added
- Procedurally generated dungeons across three floors.
- Combat: sweeping sword arc (`J` / `Space`) and thrown daggers (`K` / click).
- Two enemy types — crawlers (wall-bound) and ghosts (phase through walls), each with health.
- **Vault Warden** boss on floor 3 with a health bar, charge attack, and minion summons.
- Between-floor **shop**: full heal, potions, max-life, swift boots, brighter lantern, sword & dagger upgrades.
- Fog-of-war **minimap** and a **compass** arrow to the current objective.
- Dynamic torch lighting and a lantern of darkness around the player.
- Web Audio **ambient drone** and procedural sound effects; mute toggle (`M`).
- **Saved progress** via `localStorage`: continue an in-progress run, plus persistent records and run history.
- **Touch controls** (virtual stick + action buttons) and responsive scaling for phones/tablets.
- **PWA** support (`manifest.json` + service worker) for offline play and home-screen install.
- **Capacitor** project under `android/` for building a native APK.
